# FIFA Live — Public Stream

This project hosts an embedded FIFA live stream experience from the current workspace folder, FIFA_2026.

Run locally:

```bash
cd "C:\Users\nehab\Desktop\My Projects\FIFA_2026"
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```

Notes:
- The embedded content is loaded from public/fifa-live-source.html.
- Make sure any third-party assets used by the source are allowed to load in your deployment environment.
- The app is built with Vite and React for a lightweight public preview experience.
